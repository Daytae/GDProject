Guided Diffusion Project:

This is a playground for exploring guided diffusion.

**Large Files**
artifacts
- Wandb runs and other temporary / not useful folder. Not commited to github for large data reasons
data
- The datasets the diffusion models are trained on. Also not commited to github as data is large
saved_models
- The saved models for the diffusion / vae / bayesopt

datasets
- The code for managing the datasets
model
- The code for the vae / diffusion / bayesopt / etc.
scripts
- The scripts run to train / do other things on the server
train
- Trainer infastructure
util
- Utility functions for chem, visualization, statistics, etc.
